# XMPP Chat - Python

## Description

Python chat using XMPP protocol

## Features

- [x] Sign up
- [x] Login
- [x] Delete account
- [x] Logout
- [x] Add friends
- [x] Show contact list
- [x] Show contact's information
- [x] Private chat
- [x] Chat group
- [x] Update status and status message (presence)
- [x] Send and receive notifications

## Install project

1. Clone project
2. pip install -r requirements.txt
3. python main.py
